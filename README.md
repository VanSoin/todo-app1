# Simple Todo App

Open `index.html` in a browser to use the app. Features:
- Add todos via the input and `Add` button or Enter key.
- Delete todos with the `Delete` button next to each item.
- Toggle between light and dark themes using the button at the top; preference is remembered across reloads.

Files:
- index.html — UI
- styles.css — basic styling
- app.js — add/delete logic
